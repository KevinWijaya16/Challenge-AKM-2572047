Apa itu AWS Lambda?

AWS Lambda adalah layanan komputasi berbasis serverless yang disediakan oleh Amazon Web Services (AWS). Layanan ini memungkinkan Anda untuk menjalankan kode program tanpa perlu memikirkan penyediaan (provisioning) atau pengelolaan server fisik maupun virtual.

Cukup unggah kode Anda dalam bahasa pemrograman yang didukung (seperti Python, Node.js, atau Go), dan AWS Lambda akan menangani semua infrastruktur mulai dari manajemen kapasitas, penskalaan otomatis (auto-scaling), hingga pemantauan secara otomatis.

Serverless

Anda tidak perlu memikirkan penyediaan atau pengelolaan server. Cukup unggah kode Anda, dan AWS Lambda yang akan mengurus semuanya.

Event-Driven

Fungsi Anda berjalan otomatis merespons pemicu (event) dari layanan AWS lain (seperti upload file di S3 atau HTTP request via API Gateway).

Pay As You Go

Anda hanya membayar berdasarkan jumlah pemicuan dan durasi eksekusi kode dalam hitungan milidetik. Nol biaya saat kode tidak berjalan.

Bagaimana Lambda Bekerja?

4 langkah sederhana dari penulisan kode hingga eksekusi otomatis

01
Tulis Kode

Anda menyusun baris kode mandiri terisolasi yang disebut fungsi untuk menyelesaikan satu tugas spesifik.

02
Atur Hak Akses

Gunakan peran IAM eksekusi guna mengontrol secara ketat layanan AWS apa saja yang boleh berinteraksi dengan fungsi Anda.

03
Pemicu Masuk

Event/pemicu eksternal datang, lalu melemparkan data kontekstual berformat JSON langsung ke arah fungsi.

04
Eksekusi Kilat

Lambda langsung menyiapkan runtime terisolasi, mengeksekusi kode, dan mati otomatis sesaat setelah selesai.

Fitur-Fitur Utama Lambda

Variabel Lingkungan & Versi

Modifikasi perilaku aplikasi secara instan tanpa ubah kode utama, serta uji fitur baru dengan sistem rollback aman.

Lambda Layers & SnapStart

Bagikan pustaka bersama antar fungsi agar hemat ruang, dan pangkas durasi cold start awal ke performa sub-detik.

Integrasi Jaringan & Fungsi URL

Sambungkan fungsi ke sistem database privat via VPC, atau buat public endpoint langsung tanpa butuh API Gateway.

4 Komponen Inti Lambda

Komponen Peran Sederhana
Fungsi & Handler Wadah logika utama program beserta gerbang masuk eksekusi kode ketika pemicu datang.
Runtime & Environment Ruang isolasi mesin virtual AWS beserta penerjemah bahasa pemrogramannya (misal: Python/Node.js).
Event & Pemicu Sinyal bel pintu eksternal yang memerintahkan Lambda untuk langsung terjaga dan beroperasi.
Izin & Peran (IAM) Kunci otentikasi digital penentu batasan hak operasional fungsi ke resource AWS lain.
